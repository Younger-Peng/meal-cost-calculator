# 饭费计算器

省去人工计算的烦恼
![meal-calc](https://s2.ax1x.com/2019/03/25/At6j7d.jpg)

## 启动项目

```sh
npm install
```

```sh
npm start
```

## 注意事项

使用前，请将本项目中涉及到的微信联系人，按照 `bank.json` 中的名字重新设置备注

## 充值
如果您想充值，直接在 "前端小分队" 里回复 `+100`，即可充值到账。当然，这只是数字上的。

## 消费
如果您想充值，直接在 "前端小分队" 里回复 `-16`，即可从您的账户扣除 `16`。
